# Nike Landing Page

- Welcome to the Nike Landing Page project! This static landing page is built using `React` and `Tailwind CSS`, powered by `Vite`, and deployed on `Vercel`.
- The design is fully responsive, ensuring a seamless experience across all devices.
- The page includes various sections: Navbar, Hero, Popular Product, About Us, Services, Special Offer, Customer Review, Contact Us, and Footer.
- The buttons and links (except for navigation links) are non-functional and serve only as placeholders.
  
 ### **[View Live](https://nike-pi-sand.vercel.app/)**

## Technologies Used

- **React** : For building the user interface.
- **Tailwind CSS** : For styling and responsiveness.
- **Vite** : For fast development and build tool.
- **Vercel** : For deployment and hosting.
